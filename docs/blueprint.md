# **App Name**: SkillUp Interview Ace

## Core Features:

- AI-Powered Mock Interviews: Simulate realistic interviews with dynamic question generation based on job roles and industries using GPT-4o as a tool.
- Multi-Role Interview Practice: Provide customizable interview scenarios for different positions and sectors.
- Real-Time Speech Recognition: Convert spoken responses to text during mock interviews using the Web Speech API.
- AI Performance Feedback: Offer AI-generated analysis of answers with scoring, body language detection and improvement suggestions.
- Progress Tracking Dashboard: Display visual analytics showing improvement over time and skill development.
- Interview Recording & Playback: Incorporate video and audio recording capabilities for self-review and analysis.
- Personalized Learning Paths: Provide customized practice recommendations based on user performance.

## Style Guidelines:

- Primary color: Deep blue (#1E40AF), instilling trust and professionalism.
- Background color: Light gray (#F8FAFC), creating a subtle, clean backdrop.
- Accent color: Emerald green (#10B981), used for success states and positive feedback.
- Headline font: 'Inter' (sans-serif) for a modern, clean, and highly legible feel.
- Body text font: System UI (sans-serif) fonts for platform-optimized performance.
- Use professional and approachable icons related to career development and skill-building.
- Implement subtle, encouraging animations for feedback and progress updates.
